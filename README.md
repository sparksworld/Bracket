## Film data source

本项目通过[GoFilm](https://github.com/ProudMuBai/GoFilm)接入数据，如有条件可自行部署云端服务(项目中含有多种部署方案)

[网页版观影地址](https://film.fe-spark.cn/)

## Getting Started

```
flutter run
```

## Build

IOS(无企业签名，请自行签名)

```
flutter build ipa
```

ADNDROID

```
flutter build apk
```
